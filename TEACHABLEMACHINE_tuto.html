Qu'est-ce que Teachable Machine ?

Teachable Machine est un outil gratuit développé par Google qui permet de créer des modèles de machine learning sans aucune connaissance en programmation. Vous pouvez entraîner un ordinateur à  reconnaître des images, des sons ou des poses corporelles, puis utiliser ce modèle dans vos projets.

1

Accéder à  Teachable Machine

Rendez-vous sur le site officiel :

https://teachablemachine.withgoogle.com/

Cliquez sur "Get Started" pour commencer.

Astuce : Aucune installation n'est nécessaire ! Tout fonctionne directement dans votre navigateur web.
2

Choisir le type de projet

Sélectionnez le type de modèle que vous souhaitez créer :

  • Image Project - Pour la reconnaissance d'images
  • Audio Project - Pour la classification de sons
  • Pose Project - Pour la dÃ©tection de poses

Puis choisissez entre Standard (recommandé) ou Embedded (pour objets connectés).

3

Créer vos classes

Une classe représente une catégorie que votre modèle doit apprendre Ã  reconnaître.

Exemple pour un projet de reconnaissance d'objets :

  • Classe 1 : "Pommes"
  • Classe 2 : "Poires"
  • Classe 3 : "cerises"

Cliquez sur "Add a class" pour ajouter autant de classes que nécessaire. Renommez chaque classe avec un nom descriptif.

4

Collecter les données d'entraînement

Pour chaque classe, vous devez fournir des exemples d'entraînement. Plus vous fournissez d'exemples variés, meilleur sera votre modèle !

Méthodes de collecte :

  • Webcam : Maintenez le bouton "Hold to Record" pour capturer des images en temps réel
  • Upload : Importez des images depuis votre ordinateur
Recommandation : Collectez au minimum 50-100 exemples par classe. Variez les angles, l'éclairage, l'arriÃ¨re-plan et la distance pour améliorer la robustesse du modèle.
⚠ Attention : Assurez-vous que vos exemples sont bien représentatifs et diversifiés. Un modèle entraîné uniquement avec des photos dans une pièce sombre ne fonctionnera pas bien en plein jour !
5

Entraîner le modèle

Une fois vos données collectées, cliquez sur le bouton "Train Model".

Le processus d'entraînement commence immédiatement. Selon la quantité de données, cela peut prendre de quelques secondes Ã  quelques minutes.

Pendant l'entraînement : Vous pouvez voir une barre de progression. Le modèle apprend Ã  distinguer vos différentes classes en analysant les caractéristiques visuelles de vos exemples.

Paramètres avancés (optionnel) :

  • Epochs : Nombre de fois que le modèle parcourt vos données (défaut : 50)
  • Batch size : Nombre d'exemples traités simultanément (défaut : 16)
  • Learning rate : Vitesse d'apprentissage (défaut : 0.001)
6

Tester votre modèle

Dans la section "Preview", testez votre modèle en temps réel avec votre webcam ou en uploadant de nouvelles images.

Le modèle affiche pour chaque classe un pourcentage de confiance indiquant sa certitude.

Exemple de résultat :

Poommes : 95% Poires : 3% Cerises : 2%
Conseil : Si les résultats ne sont pas satisfaisants, retournez Ã  l'étape 4 pour ajouter plus d'exemples d'entraînement, surtout pour les cas oÃ¹ le modèle se trompe.
7

Exporter votre modèle

Une fois satisfait des performances, cliquez sur "Export Model".

Options d'export :

  • TensorFlow.js : Pour une intégration web (JavaScript)
  • TensorFlow : Pour Python
  • TensorFlow Lite : Pour applications mobiles (Android/iOS)

Vous pouvez télécharger le modèle ou obtenir un lien partageable pour l'utiliser directement en ligne.

  • Choisissez des classes bien distinctes visuellement
  • Utilisez un bon éclairage lors de la capture
  • Variez les exemples (angles, distances, arrière-plans)
  • Testez régulièrement pendant l'entraînement
