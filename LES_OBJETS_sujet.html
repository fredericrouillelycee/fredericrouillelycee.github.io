📚 Rappels théoriques

Qu'est-ce que le Machine Learning ?

Le machine learning (apprentissage automatique) est une méthode qui permet à un ordinateur d'apprendre à partir d'exemples, sans être explicitement programmé pour chaque situation.

Principe de base :

  1. Collecte de données : On montre à l'IA de nombreux exemples
  2. Entraînement : L'IA analyse ces exemples et en extrait des caractéristiques
  3. Prédiction : L'IA peut ensuite reconnaître de nouveaux objets jamais vus

Différence avec la programmation classique :

  • Programmation classique : On écrit toutes les règles (SI cet objet est rouge ET rond ALORS c'est une pomme)
  • Machine Learning : L'IA découvre elle-même les règles à partir des exemples

📝 Ce qu'il faut retenir

  • Une IA apprend à partir d'exemples (données d'entraînement)
  • Plus les données sont nombreuses et variées, meilleure est la généralisation
  • Une IA peut reconnaître des objets jamais vus si elle a appris les caractéristiques générales
  • L'IA ne "comprend" pas vraiment : elle identifie des motifs statistiques (formes, couleurs, textures)
  • Les biais dans les données d'entraînement créent des biais dans les prédictions de l'IA
  • L'IA peut se tromper et ne dit jamais "je ne sais pas" → danger de fausses certitudes
  • Le machine learning nécessite une réflexion éthique sur ses usages et ses limites
