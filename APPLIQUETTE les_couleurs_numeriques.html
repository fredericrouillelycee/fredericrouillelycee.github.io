💡 À savoir

Plus le nombre de bits est élevé, plus on peut représenter de couleurs différentes, mais plus l'image occupe d'espace en mémoire !

📊 Informations Techniques

Code RGB : rgb(128, 128, 128)
Code Hexadécimal : #808080
Bits par canal : 8 bits
Taille par pixel : 24 bits (3 octets)
