1
Préparation et collecte des données
⏱️ 15 min

A Découvrir l'interface du simulateur

Vous allez utiliser un simulateur de classification de texte (basé sur l'algorithme Naïve Bayes). Il se trouve plus bas dans cette page.

Dans ce simulateur, vous pouvez :

  • Ajouter des phrases dans deux catégories : PUBLIER (commentaires acceptables) et MODERER (commentaires à modérer).
  • Lancer l'entraînement pour que l'IA apprenne à partir de ces exemples.
  • Tester de nouvelles phrases et observer les probabilités.

B Créer vos deux classes

Dans le simulateur, vous avez deux catégories :

  • 🔵 PUBLIER : commentaires positifs, neutres, constructifs.
  • 🟡 MODERER : commentaires insultants, agressifs ou hors-sujet.

C Nourrir l'IA (jeu de données)

Ajoutez au moins 10 exemples dans chaque catégorie. Variez les formulations.

Vous pouvez utiliser les zones d'import ci-dessous pour copier-coller une liste de phrases (une par ligne).

2
Entraînement et premiers tests
⏱️ 15 min

A Lancer l'entraînement

Cliquez sur le bouton "🚀 Entraîner l'IA" dans le simulateur.

Observez les statistiques affichées : nombre d'exemples par catégorie, nombre de mots uniques.

B Tester votre IA

Dans la partie Test du simulateur, entrez des phrases nouvelles (jamais vues) et observez les résultats :

  • "Génial, merci pour le partage"
  • "Tu racontes n'importe quoi"
  • "Je ne suis pas d'accord mais c'est intéressant"
  • "Ta gueule"

Notez les décisions et les probabilités. L'IA se trompe-t-elle parfois ? Pourquoi ?

3
Enquête sur les biais
⏱️ 15 min

A Le problème des données déséquilibrées

Nous allons volontairement créer un jeu de données déséquilibré pour observer l'effet sur les décisions.

  1. Cliquez sur "🗑️ Tout effacer" pour repartir de zéro.
  2. Ajoutez 20 phrases variées dans PUBLIER (vous pouvez utiliser le bouton "📋 Ajouter des exemples par défaut" puis en ajouter d'autres).
  3. Ajoutez seulement 3 phrases dans MODERER (par exemple : "nul", "merde", "idiot").
  4. Entraînez le modèle.

B Observez les conséquences

Testez maintenant la phrase : "Je trouve cette idée stupide".

Que répond l'IA ? Est-ce correct ? Pourquoi ?

❓ Question : Pourquoi l'IA a-t-elle du mal avec cette phrase ? Que faudrait-il faire pour améliorer ses performances ?

C Comprendre le biais (en groupe)

Discutez avec votre binôme et répondez aux questions suivantes (vous les noterez sur la fiche de rendu) :

  1. Pourquoi l'IA a-t-elle du mal avec les nouvelles insultes ?
  2. Que faudrait-il faire pour améliorer ses performances ?
  3. Imaginez qu'on ait entraîné cette IA uniquement avec des commentaires d'hommes adultes. Quel pourrait être le problème quand elle modérera un forum utilisé par des adolescents ou des femmes ?

➡️ Utilisez le simulateur ci-dessous pour réaliser les manipulations.
