🧠 TP : Créer un prompt pour générer une page web de vente de chaussures

Objectif : Rédiger un prompt détaillé à destination d’une IA (ex. ChatGPT, Copilot, Mistral) afin qu’elle génère une page d’accueil (landing page) pour un magasin de chaussures.

⭐ Page de référence : Avant de commencer, ouvrez cette page d'exemple (fichier facade.html) dans votre navigateur. C'est le type de résultat que l'on souhaite obtenir. Vous pouvez vous en inspirer, mais votre prompt doit être original et décrire précisément la page que vous voulez voir générer.

Vous allez construire votre prompt étape par étape. À la fin, vous rassemblerez toutes vos idées dans un prompt final, prêt à être testé (si vous le souhaitez, vous pourrez le copier dans une IA pour voir le résultat).

📝 Étape 1 – Description générale et cible

Avant de parler technique, décrivez l'identité du magasin : nom, ambiance, public visé, message principal.

❶ Quel nom donnez-vous à votre magasin de chaussures ? (ex. SneakerSpot, Urban Feet…)
❷ Quelle ambiance souhaitez-vous dégager ? (ex. dynamique/jeune, luxe/minéraliste, éco-responsable, vintage…) Donnez 3 adjectifs.
❸ Qui est votre client type ? (âge, style de vie, centre d’intérêt)

🧱 Étape 2 – Structure de la page

Découpez la page en sections (en-tête, corps, pied de page). Soyez précis sur l’ordre et le rôle de chaque bloc.

❶ Énumérez les sections principales, de haut en bas (ex. : barre de navigation, bannière hero, grille de produits, newsletter, footer…).
❷ Pour la section hero, que doit-elle contenir ? (titre, sous-titre, bouton, image…)
❸ Combien de produits souhaitez-vous afficher dans la grille ? Faut-il un bouton « ajouter au panier » sur chaque carte ?

🎨 Étape 3 – Style visuel

Définissez les couleurs, la typographie, les formes (bords arrondis, ombres, espacements).

❶ Palette de couleurs : donnez une couleur principale (boutons, liens) et une couleur de fond.
❷ Quelle police d'écriture ? (si vous n'avez pas de nom, décrivez le style : moderne avec empattement, sans serif, etc.)
❸ Détails de design : bords arrondis ou carrés ? ombres ? transitions ?

📸 Étape 4 – Contenu (textes et images)

Rédigez les textes (titres, accroches) et indiquez le type d'images souhaitées.

❶ Proposez un titre accrocheur pour la section hero.
❷ Rédigez un court slogan ou paragraphe pour accompagner le titre.
❸ Description des images : que doivent représenter les images ? (photos de chaussures, mannequins, ambiance urbaine…). Préférez-vous des images libres de droit (Unsplash, Pexels) ?

🚀 Étape 5 – Rédaction du prompt final

En utilisant toutes vos réponses, écrivez un prompt unique, clair et structuré, que vous pourriez donner à une IA. Pensez à être précis et à inclure toutes les informations importantes (structure, style, contenu).

✏️ Votre prompt final (vous pouvez l'écrire en français ou en anglais, mais soyez cohérent).

Conseil : relisez-vous et vérifiez que vous n'avez rien oublié (en-tête, responsive, appel à l'action, footer…).
