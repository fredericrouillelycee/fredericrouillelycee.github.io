Tutoriel – Utilisation de l’application IA

Reconnaissance de formes géométriques

@ F.Rouillé - 2026

🎯 Objectif

Comprendre comment une IA apprend à reconnaître des formes simples.

1️⃣ Dessiner une forme

- Utiliser le mode Dessin libre ou Polygone
- Vous pouvez tracer des formes parfaites (carré, triangle, cercle)
- Le quadrillage aide à placer les sommets

2️⃣ Entraîner l’IA

1. Dessinez une forme propre
2. Cliquez sur Apprendre (triangle, carré ou cercle)
3. Répétez pour chaque type de forme

3️⃣ Choisir les paramètres

Vous pouvez sélectionner les critères utilisés par l’IA :
  • Nombre de sommets
  • Ratio largeur / hauteur
  • Surface
Testez différentes combinaisons !

4️⃣ Reconnaître une forme

Dessinez une nouvelle forme puis cliquez sur Reconnaître. L’IA compare les caractéristiques numériques.

5️⃣ Tester le mode bruit

Activez Mode bruit pour simuler des imperfections. Observez l’influence sur la reconnaissance.

🧠 À retenir

Une IA ne "voit" pas une forme. Elle compare des nombres calculés à partir du dessin.
